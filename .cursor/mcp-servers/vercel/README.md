# Vercel MCP

Official remote MCP — no local install. Configured in [`.cursor/mcp.json`](../../mcp.json).

## Setup

1. Restart Cursor or refresh MCP (Settings → Tools & MCP).
2. Click **Needs login** on the Vercel server and authorize with your Vercel account.

## What you can do

- List projects and deployments
- Inspect failed builds and fetch logs
- Manage env vars (per Vercel MCP permissions)

Docs: [Vercel MCP](https://vercel.com/docs/agent-resources/vercel-mcp)

Your Rasoi API is at `https://rasoiapplication.in` (also `https://rasoiapp.vercel.app`).
